Data comes from parsing the latest VAERS data file 2021VAERSDATA.csv from
https://vaers.hhs.gov/data/datasets.html
The columns on the file are
'VAERS_ID', 'RECVDATE', 'STATE', 'AGE_YRS', 'CAGE_YR', 'CAGE_MO', 'SEX', 'RPT_DATE', 'SYMPTOM_TEXT', 'DIED', 'DATEDIED', 'L_THREAT', 'ER_VISIT', 'HOSPITAL', 'HOSPDAYS', 'X_STAY', 'DISABLE', 'RECOVD', 'VAX_DATE', 'ONSET_DATE', 'NUMDAYS', 'LAB_DATA', 'V_ADMINBY', 'V_FUNDBY', 'OTHER_MEDS', 'CUR_ILL', 'HISTORY', 'PRIOR_VAX', 'SPLTTYPE', 'FORM_VERS', 'TODAYS_DATE', 'BIRTH_DEFECT', 'OFC_VISIT', 'ER_ED_VISIT','ALLERGIES'
For clarity the report includes only these columns
'VAERS_ID','RECVDATE','SYMPTOM_TEXT','STATE','AGE_YRS','DIED'
The conditions for data to show in the report are:
DIED must be Y
SYMPTOM_TEXT must include the string 'covid' or 'coronavirus' (any case)
